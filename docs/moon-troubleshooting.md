# Moon Setup Troubleshooting (Archived)

The Moon UI has been retired and replaced by the OnceUI-powered deployment wizard served from `deployment/panel/`. Use the new dashboard for install/start/stop flows, live NDJSON logs, and settings management.

For current guidance, follow the deployment panel steps outlined in [docs/deployment.md](./deployment.md) and the end-to-end OnceUI walkthrough in [docs/onceui-deployment-wizard.md](./onceui-deployment-wizard.md). Those documents cover bootstrap requirements, build commands (`npm run deploy:panel:build`), endpoint mappings, and the streaming output surfaces available in the OnceUI template.
